## 3. 2진수 덧셈 및 결과

a. 10101111 + 11011011  
10101111 = 175  
11011011 = 219  
175 + 219 = 394  
394 → 2진수: 110001010  

b. 10010111 + 11111111  
10010111 = 151  
11111111 = 255  
151 + 255 = 406  
406 → 2진수: 110010110  

c. 01110101 + 10101100  
01110101 = 117  
10101100 = 172  
117 + 172 = 289  
289 → 2진수: 100100001
